# WalkQuote – Streamlit Web App

WalkQuote is a simple and visually appealing web application built using *Python* and *Streamlit*.  
The app features custom page configuration, gradient-based UI styling, and automatic dependency handling for easy deployment.

##  Features
- Streamlit-based interactive web interface
- Custom page title, icon, and layout
- Gradient background and styled sidebar using CSS
- Automatic generation of requirements.txt
- Ready for deployment on Streamlit Cloud

##  Tech Stack
- Python
- Streamlit
- HTML / CSS
